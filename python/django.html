Python ☞ Django

  1. 创建model流程[创建 Question为例]

    • 使用startapp创建模型

            python manage.py startapp Question

    • 在setting.py中声明 “Question”

        找到setting.py 中的 INSTALLED_APPS 在末尾添对应的 model名称：Question

    • 使用migrate命名迁移数据库版本

        class QQuestion(models.Model):
          question_text = models.CharField(max_length=200)
           pub_date = models.DateField('date publised')

  class QChoice(models.Model):
          question = models.ForeignKey(QQuestion, on_delete=models.CASCADE)
          choice_text = models.CharField(max_length=200)
          votes = models.IntegerField(default=0)

  python manage,py migrate （验证有效性）

  完成版本的迁移和位置数据库表的创建

    • 查看对应的模型的sql语句

        python manage.py sqlmigrate Question(‘对应的模型’) 0001
